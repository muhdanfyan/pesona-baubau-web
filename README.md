# 🌴 Pesona Baubau Digital Hub — Portal Publik (PWA)

Aplikasi pariwisata mobile-first PWA untuk wisatawan domestik & mancanegara.

## 🚀 Tech Stack
- React + Vite + PWA
- Tailwind CSS
- Leaflet.js untuk peta interaktif
- Pannellum untuk virtual tour 360°

## 📁 Acuan Desain
File desain dari Google Stitch ada di `public/acuan-desain/` — gunakan sebagai referensi visual.

## 🏗 Cara Run
```bash
npm install
npm run dev
npm run build  # untuk build produksi
```
